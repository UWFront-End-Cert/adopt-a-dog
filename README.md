# Adopt a Dog

### Adopt a dog is a responsive and mobile-friendly e-commerce website built to offer a list of dogs that are looking for new homes. Users can see dog photos, description, and the cost of adoption.

## Technologies Used

---

- JavaScript
- jQuery
- HTML
- CSS
